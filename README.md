# Meu primeiro repositório!
Este é o meu primeiro projeto que enviei para o Github, nas aulas do curso de Desenvolvedor Front-End Senac São Carlos 2026.

## Como enviar projetos para o Github?
- Precisa ter o **Git** e o **Github Desktop** instalados;
- Abrir o **CMD** na pasta do projeto e executar o comando: `git init `;
- Fazer o **commit** no Github Desktop e depois clicar em `push` para enviar;
    - Se é a primeira ve, precisa clicar em `publish`;
    - Lembrar de, preferêncialmente, deixar o projeto como **public**;

## Como baixar projetos?
-  Abrir o **CMD** e digitar `git clone [link do repositório]`;
